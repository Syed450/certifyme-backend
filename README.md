# CertifyMe Backend (Flask)

## Overview
Backend implementation for CertifyMe Admin Portal using Flask.

## Features
- Admin Signup & Login
- Add Opportunity
- View Opportunities
- SQLite database integration

## How to Run
pip install -r requirements.txt  
python app.py

## API Endpoints
- POST /signup  
- POST /login  
- POST /opportunity  
- GET /opportunity/<user_id>
